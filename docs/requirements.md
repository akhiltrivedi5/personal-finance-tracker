Project documentation

## 1. Objective
Build a single-user, offline-first personal finance tracker to record income, expenses, and investments.

## 2. In Scope (MVP)
- Manual transaction entry
- Income, expense, and investment tracking
- Category and subcategory classification
- Monthly summaries
- Basic charts

## 3. Out of Scope (For Now)
- Bank integrations
- Multi-user support
- Cloud sync
- Tax calculations

## 4. User Stories
- As a user, I can add a transaction with amount, date, category, and notes.
- As a user, I can view all transactions in a list.
- As a user, I can see monthly income vs expenses.
- As a user, I can see category-wise spending.

## 5. Non-Functional Requirements
- Works offline
- Runs on PC, Mac, and mobile
- No paid services
- Data stored locally and securely
